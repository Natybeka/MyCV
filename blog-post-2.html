What is meant by default 16px font-size

By Natnael Bekabtu in Desktop. Updated on March 15,2020


Pixel units are the most common absolute unit used to size items in a viewport (available window). px values are exactly intrepreted by the browser as they are. pixel is an absolute size unit meaning that it doesn't take into consideration the scalability of the web design when it is used across different devices with different screen sizes and resolutions, not to mention the need for the user to use the zoom feature of a browser for better visibility. Other absolute units include cm, mm and pt. px length unlike realtive measurements units are not based on anything found on a webpage, buth rather on known physical measurements.


The css font-size of a font is the height of an imaginary design box around a capital "M" letter. Browsers usually have a defult font size of 16px. This means that any text written in the html body, provided that additional styling is not done, would measure to 16px in height. setting the font-size to 200px would result in increse to the text-body.
